这个.txt文件是对本程序的功能及作用的说明：
这个程序用于目标追踪，主要算法是用了高斯背景模型提取目标前景具体过程如下：
1、读入视频（可以是本地视频，也可以是摄像头读入的视频）后，
   利用BackgroundSubtractorMOG2类（这个类在D:\opencv\sources\samples\cpp\bgfg_segm.cpp）
   定义一个对象对前景进行提取；
   提取前景的时候将原始帧缩小至原来的一半，以加快处理速度！
2、提取出的前景进行滤波和形态学滤波去除各种噪声和空洞；
3、查找处理好的前景的轮廓，并对轮廓按面积大小从大到小进行排序；
4、剔除面积较小的轮廓，获得剩余轮廓的外接矩形；
5、由上面获得的矩形计算矩形的中心点作为Kalman滤波器追踪的初始化值；
6、创建一个Kalman滤波器后，将矩形的中心坐标初始化Kalman滤波器的状态（状态选为：矩形中心坐标x,y和坐标的变化值detax,detay）；
7、将矩形的中心点坐标也作为预测的初始条件（根据这两个坐标点来预测点的运动）；
8、根据预测的中心点的坐标得到新矩形左上角点的坐标，矩形的宽和高不变；
9、将得到的矩形绘制到原始帧中；
   这一步是将在缩小的帧中找到的目标再还原到原始帧的大小再绘制到原始帧中！
10、程序运行结束！

这里的Kalman滤波器追踪的程序，用到单目高斯背景模型&光流，双目追踪中用法是一样；
只是把原来的CamShift算法换成Kalman滤波器，或者把Kalman滤波器换成CamShift算法；
要替换的程序在程序中的注释里面已经标出来了。
